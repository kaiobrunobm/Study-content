Continuação das formatações

Texto marcado

Com essa tag podemos usar um "marca texto" nosso texto, com a tag <mark>

Texto grande e pequeno

Nesse paragrafo será criado um texto grande e um texto pequeno com a tag <big> para texto grande.

E com a tag <small> para texto pequeno.

Texto deletado

Nesse texto você não devera considerar já que está com a tag <del>

Texto inserido

Já nesse texto podemos marcar com a tag <ins> para considerarmos recentemente depois de adicionado

existe tambem a tag <u> mas já está quase em desuso.

Texto sobrescrito

PalavraSão tag para adicionar textos pequenos em cima de outro como nas formulas matematicas, sendo essa tag <sup>

Texto subescrito

PalavraSão tags para adicionar textos pequenos abaixo da palavra como em formula quimica sendo essa tag <sub>
